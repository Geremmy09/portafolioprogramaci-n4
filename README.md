## 📝 Sobre el proyecto
El objetivo de esta actualización fue crear una sección de **Notas o Comentarios** donde los usuarios puedan interactuar*.

### ¿Qué hace la página?
He implementado un sistema **CRUD** completo para las notas:
* **Crear:** Puedes escribir tu nombre, correo y una nota.
* **Leer:** Las notas se guardan y se muestran automáticamente abajo con la fecha en que se escribieron.
* **Editar:** Si te equivocas en algo, puedes darle al botón de editar y corregir la nota.
* **Eliminar:** Puedes borrar las notas que ya no quieras que aparezcan.

## 🛠️ Tecnologías que usé
* **HTML y CSS:** Para toda la estructura y el diseño oscuro.
* **PHP:** Para conectar la página con el servidor y manejar la lógica.
* **MySQL:** Para guardar todas las notas de forma permanente.
* **XAMPP:** Lo usé como servidor local para las pruebas.

## ⚙️ Cómo hacerlo funcionar
Si quieres probarlo en tu PC:
1. Descarga el repositorio y ponlo en tu carpeta `htdocs` de XAMPP.
2. Abre **phpMyAdmin** y crea una base de datos llamada `portafolio_geremmy`.
3. Importa el archivo `database.sql` que dejé aquí en la carpeta.
4. Abre en tu navegador: `http://localhost/nombre-de-tu-carpeta/`
